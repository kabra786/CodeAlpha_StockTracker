# 📊 Stock Portfolio Tracker

## 📌 Description
This project is a GUI-based stock tracker built using Python.
It allows users to add stocks, calculate total investment, and save results.

## 🛠️ Technologies Used
- Python
- Tkinter

## 🚀 Features
- Add stock and quantity
- Calculate total investment
- Save data to file

## ▶️ How to Run
Run the Python file:
python stock_tracker.py
